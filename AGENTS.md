# 大飞书法官网项目

## 1. 项目概述

**项目名称**: 大飞书法官网  
**项目类型**: 企业官网/教育机构官网  
**核心功能**: 展示书法教育品牌、课程体系、师资团队，提供在线咨询和试听预约  
**目标用户**: 3-12岁儿童家长、成人书法爱好者

## 2. 品牌信息

- **品牌名**: 大飞书法 / 大飞有墨
- **主校区**: 济南·海信龙奥九号
- **核心数据**: 13年教学沉淀 | 50万+家庭信赖 | 98%满意度 | 95%学员进步率
- **价值主张**: 正姿养习惯，轻松守卷面

## 3. 目标用户画像

| 用户群体 | 年龄段 | 核心痛点 |
|---------|--------|---------|
| 低龄段 | 中班-2年级 | 趴桌勾腕、写字喊累、抵触书写 |
| 高龄段 | 3年级以上 | 作业潦草、书写缓慢、卷面丢分 |
| 成人 | 职场/家长 | 职场书写潦草、自学难坚持、渴望静心 |

## 4. 页面结构

### 4.1 首页模块
1. **Hero首屏区** - 品牌Slogan、核心数据展示、CTA按钮
2. **痛点洞察区** - 分龄痛点展示、黄金习惯期概念
3. **核心优势区** - VS普通练字班对比（6大优势）
4. **课程体系区** - 硬笔/毛笔/成人/集训营展示
5. **师资团队区** - 明星教师展示
6. **学员见证区** - 真实案例展示
7. **八大保障区** - 品牌承诺展示
8. **咨询转化区** - CTA按钮、联系方式

### 4.2 其他页面
- 课程中心页
- 师资团队页
- 关于我们页
- 联系我们页

## 5. 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4

## 6. 设计规范

### 6.1 色彩体系
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主色-墨绿 | #1a5d4c | 品牌主色、导航背景 |
| 深墨色 | #1a1a2e | 首屏背景、重点文字 |
| 朱砂红 | #c23a3a | 强调色、数据展示 |
| 宣纸米 | #faf9f6 | 浅色背景、卡片背景 |
| 墨黑 | #2d2d2d | 正文文字 |
| 灰墨 | #6b7280 | 次要文字 |

### 6.2 字体规范
- 标题：思源宋体/Noto Serif SC（书法气质）
- 正文：思源黑体/Noto Sans SC（清晰易读）
- 品牌名：书法字体（体现书法特色）

### 6.3 布局规范
- 最大宽度：1400px
- 响应式断点：mobile(768px) / tablet(1024px) / desktop(1280px)
- 模块间距：80px-120px

## 7. 开发命令

```bash
pnpm install     # 安装依赖
pnpm dev         # 启动开发服务器（端口5000）
pnpm build       # 构建生产版本
pnpm lint        # 代码检查
```

## 9. 注意事项

- 所有页面必须支持响应式设计
- 图片资源使用 public 目录
- 遵循 Tailwind CSS 4 规范
- 使用 shadcn/ui 组件保持一致性
- 首屏加载性能优化（图片懒加载、CSS压缩）
